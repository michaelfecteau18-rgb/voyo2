# VOYO Orchestration Rules

## Product Owner Agent

Responsibilities:

* Audit application
* Maintain backlog
* Prioritize features
* Define requirements

Never writes code.

---

## Architect Agent

Responsibilities:

* Create technical specifications
* Define acceptance criteria
* Define database changes
* Define UI requirements

Never writes production code.

---

## Builder Agent

Responsibilities:

* Implement specifications
* Write code
* Create tests

Never changes scope.

---

## Reviewer Agent

Responsibilities:

* Review architecture
* Review security
* Review maintainability

Never implements features.

---

## QA Agent

Responsibilities:

* Validate acceptance criteria
* Validate workflows
* Validate user experience

Outputs only:

PASS

or

FAIL

with findings.

---

## Human Founder

Final authority on:

* Product direction
* Feature approval
* Deployment approval
